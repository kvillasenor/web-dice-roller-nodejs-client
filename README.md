# Server Dice Roller with Node.js - Client
Project created by Karen Villasenor

This project was made with the help of using the template provided and with the use Google AI Overview and ChatGPT. All parts were tested to make sure they work and to understand what each part does for this project. Parts of the HTML and the CSS from the previous Dice Roller assignment were reused for this assignment. The random number generator equation was also reused.

# Compiling and Executing
To run the project, download or clone the repository, make sure all of the files are in the same folder, and then open the “index.html” file in a web browser. Client is also hosted on Azure: https://red-plant-018861b10.1.azurestaticapps.net/
